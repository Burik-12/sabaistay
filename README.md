# SabaiStay 🏝️

Нейтральный агрегатор-поисковик жилья на острове **Ко Пханган**. Собирает объявления
об аренде/продаже из **Facebook** (главный источник) и **Telegram**, приводит к единому
формату и отдаёт через **витрину с фильтром + картой** и **бота с подпиской на фильтр**.

> **Зачем:** в Facebook Marketplace почти нет фильтрации, в Telegram объявления тяжело
> искать. Ценность продукта = **нормальный фильтр** (район / цена / спальни / даты /
> бассейн) поверх всего этого хаоса. Каждое объявление ведёт ссылкой на первоисточник.

## Пайплайн

Один конвейер от сырого поста до карточки в витрине. Большинство звеньев **уже сделано и
работает бесплатно** на демо-данных; платные звенья (FB-сбор, VPS, API-ключ) включаются по
мере бюджета и не меняют остальную трубу.

```text
   Facebook  (главный источник, по объёму)      Telegram  (постоянный мониторинг)
        │                                                │
        └────────────────────┬───────────────────────────┘
                             ▼
        ingest/   — сбор сырых постов  →  raw_post
            • t.me/s превью без аккаунта ........ 🟢 готово, проверено на живых данных
            • Telethon userbot 24/7 ............. 🟡 заготовка · ждёт VPS ~$5/мес + аккаунт
            • Apify + тайский резидентный прокси  🔴 чек-лист · ~$60–110/мес (FB)
                             ▼
        parser/   — сырьё  →  фильтруемые поля
            • parse.py — разбор через Claude ..... 🟡 заготовка · ждёт ANTHROPIC_API_KEY
            • normalize_district — район→канон ... 🟢 готово (11/11, exact+fuzzy)
            • dedup — склейка одного дома FB+TG .. 🟢 готово (тест TG+FB → 1 карточка)
            • freshness — active/rented/expired .. 🟢 готово (авто-expire 45 дн)
            • benchmark — цена к рынку (Airbnb) .. 🟢 готово (пока сид-медианы)
                             ▼
        db/schema.sql   — Postgres / Neon ........ 🟢 free tier (PDPA-safe: факты+ссылка)
                             ▼
            ┌────────────────┴─────────────────┐
            ▼                                   ▼
     web/  витрина                       bot/  подписка на фильтр
     фильтр + карта + бенчмарк           пуш «новый дом под твой фильтр»
     🟢 готово (самодостаточный html)    🟢 движок готов · ждёт токен + поток
```

Легенда: 🟢 бесплатно и сделано · 🟡 дёшево, ждёт ключ/аккаунт · 🔴 основная статья расходов (FB).

## Связь с Right Way

Продукт **отдельный** и **не брендирован** как RW — это сознательно (нейтральный
агрегатор всех, без конфликта интересов). Код-ядро **скопировано** из наработок RW
(витрина `/listings`, газеттир районов, парсер на Claude, коллекторы рынка), но
**живой связи нет**: отдельный репозиторий, отдельная база, отдельный домен.
Приватные данные RW сюда не попадают.

## Карта проекта

```text
docs/      — архитектура, дорожная карта, нейминг, юридика (PDPA), дизайн-система
data/      — газеттир районов (канон+алиасы+коорд), демо-выборки, сид-медианы рынка
db/        — схема базы (объявление, подписки, дедуп, бенчмарк)
ingest/    — сборщики источников (TG-превью 🟢, Telethon 🟡, FB-чек-лист 🔴)
parser/    — разбор и обогащение: parse · normalize · filter_match · dedup · freshness · benchmark
web/       — генератор витрины (build.py → самодостаточный index.html) + лого brand.svg
bot/       — Telegram-бот «подписка на фильтр»
```

Запустить локально (всё бесплатно, без ключей и сети):

```bash
python3 web/build.py              # пересобрать витрину → web/index.html (открыть двойным кликом)
python3 bot/subscribe_bot.py --demo   # офлайн-демо разбора запроса и подбора
python3 parser/dedup.py           # самотесты движков (dedup / freshness / filter_match / benchmark)
```
