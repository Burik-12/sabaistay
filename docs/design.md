# Дизайн-система SabaiStay

Направление: **«морская карта Пхангана»**. Продукт географический → карта = герой,
объявления = отметки-«глубины» на карте, цены кораллом, районы — реальными координатами
(структура несёт настоящую информацию о позиции). Сознательно мимо AI-дефолтов
(тёплый крем+сериф+терракота / чёрный+кислотный акцент / газетная вёрстка).

## Типографика
Концепция пары — **тепло места против точности прибора**: мягкий островной дисплей рядом с
строгим картографическим моноширинным.
- **Display — Baloo 2** (600 заголовки, 800 логотип): скруглённый дисплейный шрифт. Выбран под
  «сабай»-вайб (สบาย = расслабленность, мягкость): круглые формы = комфорт, а у суперсемейства
  Baloo есть тайская версия — родословная честная. Сознательно **не** Fraunces/сериф (это AI-дефолт «крем+сериф»).
- **Data/labels — Space Mono**: цены, координаты, eyebrow района, счётчики, метки фильтра, дата
  снимка. «Приборный» голос морской карты — намеренный контраст к мягкому Baloo.
- **Body — системный sans**: описания (вес не грузим вебшрифтом).

## Бренд
- **Лого-знак — «закат над водой»** (`web/brand.svg`): коралловое солнце над тремя бирюзовыми
  линиями воды (= широты морской карты, рифма к моно-координатам). Круглое солнце рифмуется со
  скруглённым Baloo. Тёмная бирюза для светлого фона/фавикона; на тёмной шапке линии — светлые.
- **Логотип — двухцветный**: `Sabai` белым + `Stay` кораллом, без разделителя.
- **Фавикон**: тот же знак, инлайн-SVG в `data:`-URI (витрина самодостаточна, без внешнего файла).
- **Голос**: тёплый, спокойный, по делу. Не туристическая попса, а «спокойно найти, где жить».

## Сигнатура
1. **Координатный eyebrow** на карточке (`Район · 9.748, 99.979`) — моно, бирюзовый.
2. **Цена-«глубина»**: коралловое моно-число, на карте — пин-пилюля с компактной ценой (`35k`), на hover увеличивается.
3. Карта на тихих тайлах CartoDB Positron → читается как чистая карта, а не громкий OSM.

## Правила
- Смелость в одном месте (карта-карта + коралловые отметки), остальное тихо и дисциплинированно.
- Коралл — только акцент (цена/CTA/пин), не заливка.
- Quality floor: адаптив до мобильного (карта сверху, список снизу), видимый фокус, `prefers-reduced-motion` уважается, анимация появления — только при первой отрисовке.

## Где живёт
Генератор `web/build.py` (шаблон + токены внутри) → `web/index.html`. Пересобрать после
изменения данных/стиля: `python3 web/build.py`. Эти токены перенести в настоящую витрину,
если будем форкать `/listings` из RW.
